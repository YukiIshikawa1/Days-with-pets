#  ペット日記

## サイト概要
<ペット達との写真を日記のように投稿できる又他の人がコメントできるサイト>

### サイトテーマ
動物達との思い出を一生に

### テーマを選んだ理由
どうしても人間に比べるとペット達の寿命が長いとは言えないので、ペット達と過ごす一生の中で色々な思い出を写真にとって保存して
その写真を共有することで同じ種類のペットを飼っている人同士で意見や質問などをしてもらえる様にして欲しいので作ろうと思いました

### ターゲットユーザ
<動物を飼っている人や動物が見たい人>

### 主な利用シーン
<撮った写真をその場でアップロードすることが出来る>
